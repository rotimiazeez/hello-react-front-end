## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- NodeJS 
- Yarn

### Setup

```bash
git clone https://github.com/rotimiazeez/hello-react-front-end
cd ./hello-react-front-end
```

### Install

```bash
npm install
```

### Usage

```bash
npm start
```
